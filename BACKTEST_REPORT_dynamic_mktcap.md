## 一、核心改进：从静态市值到动态市值池

### 问题
此前回测使用 **今日市值** 筛选 500-1000亿股票池，回头看历史表现，存在两项偏差：
- **前视偏差**：2026年市值800亿的股票，2018年可能仅50亿，当时不应在池内
- **幸存者偏差**：2018年在池但后来跌出500-1000亿的股票被排除

### 解决方案
逐月根据 **当时实际市值** 动态筛选股票池：
- 每月取最后交易日全市场市值快照（tushare `daily_basic`）
- 回测中每个信号日按所属月份的市值快照检查：500亿 ≤ 总市值 ≤ 1000亿
- 101个月覆盖（2018-01 ~ 2026-05），460,694条市值记录
